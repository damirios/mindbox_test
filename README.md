## Todo List

Стек: TypeScript, React, React testing library.

### Локальный запуск
1) Стянуть на локальную машину
2) Установить зависимости
```
npm i
```
3) Запустить приложение
```
npm start
```
Приложение доступно на http://localhost:3000

4)* Запуск тестов:
```
npm run test
```
